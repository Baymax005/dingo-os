# Branding Assets

This directory contains Dingo OS branding assets.

## Directory Structure

```
branding/
├── backgrounds/     # Desktop wallpapers
├── icons/           # Icon themes and app icons
├── logos/           # Dingo OS logos
└── plymouth/        # Boot splash themes
```

## Assets Needed

### Logos
- `dingo-logo.svg` - Main logo (vector)
- `dingo-logo.png` - Main logo (256x256)
- `dingo-icon.svg` - App icon (vector)
- `dingo-icon.png` - App icon (64x64)

### Wallpapers
- `dingo-default.png` - Default wallpaper (4K)
- `dingo-dark.png` - Dark theme wallpaper
- `dingo-light.png` - Light theme wallpaper
- Additional artistic wallpapers

### Plymouth Theme
- `dingo.plymouth` - Theme configuration
- `dingo.script` - Animation script
- Spinner/logo assets

## Design Guidelines

### Color Palette
- Primary: #E65100 (Orange)
- Secondary: #FF8F00 (Amber)
- Accent: #FFD54F (Yellow)
- Dark: #1A1A2E
- Light: #EAEAEA

### Typography
- Headlines: Fira Sans Bold
- Body: Fira Sans Regular
- Monospace: Fira Code

### Logo Usage
- Maintain aspect ratio
- Minimum clear space: 20% of logo height
- Dark logo on light backgrounds
- Light logo on dark backgrounds
