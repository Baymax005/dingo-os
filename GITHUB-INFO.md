## About (Detailed - for README section)
Dingo OS is a ready-to-use Linux distribution based on Ubuntu 24.04 LTS Noble with KDE Plasma 6 desktop. It comes pre-configured with development tools (VS Code, Docker, Python, Node.js), gaming essentials (Steam, Lutris, GameMode, GPU drivers), and blockchain development environment (Truffle, Ganache, Hardhat). Features a custom GTK4 Control Center for system management.
